# Catalogue Behaviour Specification

## Structure

-   Super departments:
    -   Tech Cornwall
    -   Agile on the Beach
-   Sub departments:
    -   Optional
    -   User can add

## Behaviour

-   User selects super department first
-   Optional sub department selection
-   Scrollable list of items
-   Minimal keyboard use

## Search

-   Secondary feature
-   Default is scroll and tap

## Item Interaction

-   Tap item
-   Enter quantity
-   Select condition
-   Add serial number if needed

## UX Principles

-   Large tap targets
-   Fast interaction
-   Minimal typing
